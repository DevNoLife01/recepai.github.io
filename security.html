1. Our Security Commitment

At RecepAI, security is fundamental to everything we do. We implement comprehensive security measures to protect your data, maintain service availability, and ensure the integrity of our AI receptionist platform.

2. Data Encryption

2.1 Encryption in Transit

  • TLS 1.3: All data transmission uses the latest TLS encryption
  • HTTPS Everywhere: All web traffic is encrypted
  • API Security: All API communications use strong encryption
  • Voice Data: Real-time voice encryption during calls

2.2 Encryption at Rest

  • AES-256: Industry-standard encryption for stored data
  • Database Encryption: All databases are encrypted at rest
  • File Storage: Encrypted file systems for all data storage
  • Backup Encryption: All backups are encrypted

2.3 Key Management

  • Hardware Security Modules (HSMs) for key storage
  • Regular key rotation policies
  • Multi-factor authentication for key access
  • Audit trails for all key operations
